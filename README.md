## 🌟 Overview

A premium, modern navigation system website for Lexus, featuring AI-powered routing, 3D city modeling, real-time traffic updates, and an interactive user experience. Built with React, TypeScript, and TailwindCSS.

### ✨ Live Demo
[View Live Demo](http://localhost:3000) - Run locally to experience

---

## 🚀 Features

### 🎯 Core Features
- **AI-Powered Routing** - Smart route suggestions based on traffic patterns
- **3D City Modeling** - Immersive maps with detailed landmarks
- **Real-Time Traffic** - Live updates with predictive analytics
- **Voice Command** - Natural language processing in 50+ languages
- **Cloud Sync** - Seamless synchronization across devices

### 🎨 Pages
- **Home** - Landing page with hero section and features overview
- **Features** - Detailed feature breakdown
- **Navigation** - Interactive navigation demo
- **Specs** - Technical specifications
- **Gallery** - Image gallery with lightbox
- **Pricing** - Subscription plans with AI assistant
- **FAQ** - Frequently asked questions with search
- **Contact** - Contact form and information
- **Login/Signup** - User authentication pages

### 🤖 AI Integration
- 24/7 AI chat support via [getchatsupport.live](https://getchatsupport.live/)
- Floating AI assistant button on all pages
- Context-aware help throughout the site
